# ❤️ Heart Stroke Prediction

A machine learning web app that predicts the risk of 
heart stroke based on health parameters.

## 🔗 Live Demo
👉 [Click here to view the app](https://heart---stroke---prediction-i4wkkdifp7zjgqtnu3dndb.streamlit.app/)

## 📊 Models Trained & Compared
| Model | Metric Used |
|-------|------------|
| Logistic Regression | Accuracy & F1 Score |
| K-Nearest Neighbors (KNN) | Accuracy & F1 Score |
| Naive Bayes | Accuracy & F1 Score |
| Decision Tree | Accuracy & F1 Score |
| Support Vector Machine (SVM) | Accuracy & F1 Score |

## 🏆 Best Model
The best performing model was selected based on 
highest accuracy and F1 score and saved using Joblib.
